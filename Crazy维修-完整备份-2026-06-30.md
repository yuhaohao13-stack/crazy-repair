## 八、恢复步骤

```bash
# 1. 克隆代码
git clone https://github.com/yuhaohao13-stack/crazy-repair.git
cd crazy-repair

# 2. 安装依赖
npm install

# 3. 本地开发
npm run dev

# 4. 部署到 Vercel
# GitHub → Vercel Import → 自动部署

# 5. 域名配置
# 阿里云 → DNS解析 → CNAME → cname.vercel-dns.com
# Vercel → Settings → Domains → crazy-repair.com + www.crazy-repair.com
```
